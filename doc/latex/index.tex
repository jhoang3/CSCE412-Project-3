\chapter{Load Balancer Simulation }
\hypertarget{index}{}\label{index}\index{Load Balancer Simulation@{Load Balancer Simulation}}
\hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Introduction}{Introduction}}\label{index_intro_sec}
CSCE 412 Project 3:\+ A simulation of a load balancer that distributes web requests across a cluster of servers. The system includes:\+


\begin{DoxyItemize}
\item {\bfseries{\doxylink{structRequest}{Request}}} generation with random IPs, processing time (cycles), and job type (P/\+S).
\item {\bfseries{\doxylink{classWebServer}{Web\+Server}}} instances that process one request at a time over multiple clock cycles.
\item {\bfseries{\doxylink{classLoadBalancer}{Load\+Balancer}}} that holds a queue of requests, dispatches to idle servers, scales the cluster up/\+down based on queue size, and applies an IP firewall.
\item {\bfseries{\doxylink{structConfig}{Config}}} loaded from a file (thresholds, cooldown, blocked IP ranges).
\item {\bfseries{\doxylink{classSwitch}{Switch}}} (bonus):\+ routes requests by job\+\_\+type to two load balancers (Streaming \textquotesingle{}S\textquotesingle{} and Processing \textquotesingle{}P\textquotesingle{}), with per-\/\+LB statistics.
\end{DoxyItemize}\hypertarget{index_usage_sec}{}\doxysection{\texorpdfstring{Usage}{Usage}}\label{index_usage_sec}
Run the program and enter:\+
\begin{DoxyItemize}
\item Use \doxylink{classSwitch}{Switch}? (1=yes, 0=no). If yes, enter streaming and processing server counts; if no, enter single web server count.
\item Run time (clock cycles)
\end{DoxyItemize}

The initial queue is filled with (servers × 100) requests (or, in \doxylink{classSwitch}{Switch} mode, requests are split by job type). Each cycle, new requests may arrive at random; the load balancer scales servers when the queue exceeds 80×servers (add one) or falls below 50×servers (remove one, if idle).
